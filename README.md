## Commit Message Convention

> 커밋태그(#이슈 번호): 내용

ex. `feat(#40) : login 파일 추가'

| command | mean |
| --- | --- |
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 수정 |
| style | 코드 포맷 변경 |
| test | 테스트 코드 추가 |
| refactoring | 코드 리팩토링 |
| chore | 빌드, 패키지 매니저 수정 |
